选择排序

2021-03-23 C++

算法原理

  • 定义变量Min储存数组中最小值的下标
  • 初始时，将数组第一个元素赋值给Min
  • Min赋值过后，定义变量 j 向后搜索 Min 后元素是否小于Min
  • 若Min后元素小于Min，则 a[Min] 与 a[j] 交换位置

参考代码


#define _CRT_SECURE_NO_WARNINGS 1
#include<stdio.h>
#define M 10
int main()
{
    int a[M], min, j, i, t=0;
    for (i = 0; i < M; i++)
        scanf("%d", &a[i]);
    for (i = 0; i < M - 1; i++)
    {
        min = i;
        for (j = i+1; j < M; j++)
            if (a[min] > a[j])
                min = j;
        t = a[i];
        a[i] = a[min];
        a[min] = t;
    }
    for (i = 0; i < M; i++)
        printf("%d\n", a[i]);
    return 0;
}
